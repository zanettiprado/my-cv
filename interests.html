Talking about how I started

My very first project

My very first project was a Book Club website developed using HTML and CSS. This webpage was created to attend the first milestone project in the code institute. It's easy to find some mistakes in CSS and also in HTML, but I had never coded before. I liked the idea of starting to study code, and now I am passionate about it. I think the mistakes and the time I spent working on this project were really valuable in my journey. You can see the live website here. After Reading book club

My first JavaScript Project

This project was amazing. I loved studying JavaScript and creating this project; it was really enjoyable and challenging. First, because I had some touches and knowledge about HTML and CSS before, but I had never used JavaScript in my life. After studying the whole module, I didn't feel prepared to create my first project, but by working hard and studying diligently, I think I have created a really nice project with some tips from a lot of documentation and videos I have watched. I hope that since this first project, I have developed my knowledge into more advanced JavaScript functions. You can find the live version here. Rock Memory Game

Working with Python

This was a big challenge - switching from JavaScript to Python in just a few days. Learning the differences between the languages was messing with my head. After a few months, I finally learned Python, but I felt like I had forgotten everything I learned in the previous JavaScript module. Nevertheless, I kept going and started thinking about what I would like to create, whether it's managing data or creating a simple game using a few functions in Python. I decided to create a battleship game to demonstrate some of my abilities with this language, which, for me, is harder than JavaScript. You can find the live version here. battleship Game
